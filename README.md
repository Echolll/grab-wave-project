# Universal Video Downloader Pro

Десктопное приложение для скачивания медиаконтента (видео и аудио) с любых платформ: YouTube, VK, RuTube, TikTok, Instagram и сотен других. Построено на базе `yt-dlp` и `PyQt6`.

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.9+
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

## Возможности (Pro-версия)
* **Поддержка сотен сайтов** (благодаря `yt-dlp`).
* **Предпросмотр**: автоматическая загрузка обложки (превью) и точного названия видео при вставке ссылки.
* **Выбор качества**: скачивание от 480p до 1080p+, а также извлечение чистого **MP3 аудио**.
* **Плейлисты**: массовое скачивание плейлистов в один клик.
* **Авто-авторизация**: обход блокировок (куки читаются с помощью теневого копирования `browser-cookie3` прямо из вашего основного браузера, даже если он открыт!).
* **История скачиваний**: двойной клик по скачанному файлу открывает папку с выделенным видео прямо в Проводнике Windows.
* **Выбор папки сохранения**.
